# ebook-store

该项目为学习Vue而创建，依照慕课网《Vue 实战商业级读书Web APP》进行练习。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
